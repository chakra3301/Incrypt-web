Incrypt: Web-Based Image Steganography Tool

===========================================

Incrypt allows you to encode secret messages inside images and decode them later using only your browser. Access the encoder app by clicking the "Encoder" button in the header.

What Is This?

Incrypt is a client-side steganography tool built with modern web technologies. It lets you:

  • Upload an image (PNG/JPG/WebP)
  • Hide a secret text message inside that image using the Least Significant Bit (LSB) method
  • Adjust bit depth settings for optimal encoding (1-8 bits per channel)
  • Optionally enable pointillist mode for a creative, obfuscated result
  • Download the result as a PNG file
  • Re-upload and decode any stego-image to reveal the hidden message
  • View real-time progress during encoding/decoding operations

All processing happens locally in your browser using Web Workers—your data never leaves your device.

Important Notice: Large Files & Aspect Ratio

⚠️ For very large files: If you're encoding large amounts of data, we recommend using images with a 1:1 aspect ratio (square images) for optimal performance and reliability. This ensures the encoding process works smoothly without timeouts or memory issues.

How To Use It

  1. Access the Encoder: Click the "Encoder" button in the header to open the encoding application.
  2. Upload an Image: Click the file upload area and select any image from your device.
  3. Enter Your Message: Type the message you want to hide in the text area.
  4. Adjust Settings (Optional): Modify the bit depth if needed (default is 1 bit per channel).
  5. (Optional) Toggle Pointillist Mode: Use the pointillist mode toggle to enable or disable the artistic effect. When enabled, the image will be stylized with colored dots over a darkened version of the original image.
  6. Click Encode: This embeds your message into the image and creates a downloadable link.
  7. Decode a Stego Image: Upload a previously encoded image and click the Decode button. The hidden message will appear in the output area. The tool automatically detects if pointillist mode was used.

How It Works

  • Your message is converted to bytes and its length is stored in the image.
  • The message is encoded into the image's pixel data using the Least Significant Bit (LSB) method with configurable bit depth.
  • Processing is handled by Web Workers to prevent browser freezing during large operations.
  • If pointillist mode is enabled, the image is first stylized with colored dots over a darkened version of the original image, then the message is embedded.
  • A 1-byte header is embedded to indicate if pointillist mode was used, so decoding is always automatic and reliable.
  • Real-time progress indicators show encoding/decoding status.

Output

  • The result is a PNG image with your message embedded.
  • Anyone with this tool can decode the message, regardless of the mode used.

Privacy & Security

  • Completely client-side
  • No data is sent to any server
  • Works offline once loaded

Tips

  • For stealth, use normal mode (pointillist off) for minimal visual change.
  • For obfuscation, use pointillist mode for a creative, artistic look that disguises the message.
  • Use lower bit depths (1-2 bits) for maximum stealth and compatibility.
  • Use higher bit depths (3-8 bits) for larger message capacity, but be aware this may create more visible artifacts.
  • Avoid overly compressed formats like JPG for encoding, as compression can damage hidden data.
  • For large messages, consider using square images (1:1 aspect ratio) for optimal performance.
  • The app automatically handles timeouts and provides progress feedback for long operations.
