Incrypt: Web-Based Image Steganography Tool

===========================================

Incrypt allows you to encode secret messages inside images and decode them later using only your browser.

What Is This?

Incrypt is a client-side steganography tool. It lets you:

  • Upload an image (PNG/JPG)
  • Hide a secret text message inside that image using the Least Significant Bit (LSB) method
  • Optionally convert your image into a pointillist-style painting to disguise the data more creatively
  • Download the result
  • Re-upload and decode any stego-image to reveal the hidden message

All processing happens locally in your browser—your data never leaves your device.

How To Use It

1. Upload an Image

Click the "Choose File" button and select any image from your device. Recommended size: square or near-square.

2. Enter Your Message

Type the message you want to hide in the textarea below the file input.

3. (Optional) Toggle Advanced Settings

Click the "Advanced Settings" button to reveal the following options:

Setting Description
Dots Controls how many dots are used in pointillist mode (higher = more detail)
Jitter Adds randomness to the color of each dot (higher = noisier look)
Bit Depth How many LSBs to use for encoding (1–8). Higher stores more but may be more visible
Pointillist Mode Enables dot-painting style instead of pixel-perfect rendering

4. Click Encode

This embeds your message into the image and creates a downloadable link.

5. Decode a Stego Image

  • Upload the previously encoded image
  • Click the Decode button
  • The hidden message will appear in the output box

How It Works

Encoding:

  • Text is converted into UTF-8 bytes
  • Length (16-bit) is prepended
  • Message is encoded into image pixel data using the Least Significant Bit (LSB)
  • Optionally, the image is redrawn in a pointillist style with noise

Decoding:

  • Image pixels are read and LSBs extracted
  • First 16 bits determine the message length
  • Bits are grouped into bytes and decoded into text

Settings Explained

Setting Range Default Notes
Dots 100–20000 5000 Only applies in pointillist mode
Jitter 0–30 6 Adds randomness to colors
Bit Depth 1–8 1 Higher bit depth = more capacity
Pointillist Mode On/Off Off Uses artistic dot rendering

Output

  • Result is a PNG image with your message embedded
  • Can be shared and decoded by anyone with this tool

Privacy & Security

  • Completely client-side
  • No data is sent to any server
  • Works offline once loaded

Tips

  • For stealth: use bit depth 1–2 and disable pointillist mode
  • For obfuscation: enable pointillist mode with high jitter
  • Avoid overly compressed formats like JPG for encoding
